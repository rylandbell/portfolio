Playlist Dashboard for Spotify

React
Redux

Spotify algorithmically computes numerical values to describe song features like danceability, instrumentalness, and popularity. Playlist Dashboard lets users:

  • Load playlists from their own Spotify accounts.
  • Visualize a playlist by plotting any combination of audio features against a track's position in the selected playlist.
  • Refine a playlist by filtering or sorting with respect to any of these features.
  • Save the resulting playlist to their Spotify accounts.

Pigeon Hoops: An NBA Predictions Game

React
Node
Express
MongoDB

Pick the winner of one NBA game per day. Over the course of a month, don’t pick the same team twice. The player with the most correct picks at the end of the month wins.

I’ve written a full-stack web app to implement the above game. On the back end, an Express app works with a MongoDB database to track user picks, enforce the each-team-once-per-month rule, and score predictions using game outcome data from nba.com. In the browser, a React/Redux app provides a user-friendly interface for making picks and tracking outcomes over the course of a month.



Recovery Center Client Tracking

Node
Express
jQuery
Google Calendar API

I worked with a small team to design an app to help substance abuse recovery facilities manage their work with clients. It includes features like:

  • Displaying daily check-in data that clients have submitted through a separate app, covering things like sleep quality or cravings level.
  • Tracking basic client details like contact info and administrative status.
  • A simple database for storing clinician's session notes while complying with HIPAA.
  • A scheduling tool to display Google calendars and give clinicians an easy way to block off appointment time slots, which are then booked by clients through the youcanbook.me service.
